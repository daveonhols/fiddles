red-star-pass

Bootstrap proof of concept for an advanced password input mechanism.

Clicking icons will add them to the start of your password. Click added icons to remove them.

Password bytes
[]

Users can choose up to three custom glyphs to pre-prend to their password to increase entropy and defeat dictionary driven attacks.

Glyphs are mapped to non-printable ASCII bytes in the range 1-21 and should be salted and hashed on the server side in the usual way.

Enhancements include shuffling of the input glyphs to defeat click point analysis and show / reveal functionality to defeat screen shot attacks.
